# Конструктор меню

Оффлайн desktop-приложение Mad Marathon для сборки меню, локальной истории и выгрузки готовых PNG.

## Сборка

```powershell
npm install
npm run build
```

Готовый продукт появляется в `dist/Готовый продукт`:

- `Конструктор меню.exe`
- `templates/`
- `history/`
- `saved-menus/`
- `update-source.json`

Папки `history` и `saved-menus` являются пользовательскими данными. Их нельзя удалять или заменять при обновлении.

## Релизы и автообновление

Приложение проверяет обновления через GitHub Releases:

```text
https://api.github.com/repos/maxzoa/mad/releases/latest
```

Для новой версии нужно создать release с тегом выше текущего, например `v2.0.2`, и прикрепить asset `Конструктор меню.exe`.

При запуске приложение само скачает новый exe, заменит только exe-файл и перезапустится. Папки `history`, `saved-menus` и пользовательские шаблоны не трогаются.
